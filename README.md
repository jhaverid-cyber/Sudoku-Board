## Project title: Dhiren's Sudoku Board
## Student name: Dhiren Jhaveri
## Short description of the project
This program builds a solvable sudoku board in a console Java application in CodeHS.
## How to run the program:
The program must be run in CodeHS in the Java Console. This does not work as a stand alone java program and must be run in codeHS. 
## Summary of how the Sudoku board is generated
The Sudoku Board is generated from creating a 9x9 2D array. Each row is built one at a time of an array list of numbers 1-9. Before placing each value, every row, every column, and every box must be checked for the value to be added for the sudoku board to work. The Board is then printed.
